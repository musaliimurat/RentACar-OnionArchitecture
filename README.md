#  RentACar – Tədris məqsədli ASP.NET Core layihəsi (Onion Architecture + CQRS)

Salam əziz tələbələr! 
Bu layihə ASP.NET Core öyrənən və **peşəkar səviyyədə proqram təminatı qurmaq** istəyən hər bir tələbə üçün hazırlanıb. Layihədə **modern arxitektura prinsipləri**, **təmiz kod**, **testləşdirmə** və **real dünya layihə strukturu** nümayiş etdirilir.

Əsas məqsədimiz sizə aşağıdakıları öyrətməkdir:

- Enterprise səviyyəli **Onion Architecture** ilə işləməyi
- **CQRS (Command Query Responsibility Segregation)** pattern-i
- **MediatR**, **AutoMapper**, **FluentValidation** və digər əhəmiyyətli alətlərin istifadəsi
- Layer-based tətbiq strukturu və **modulyar dizayn**
- RESTful API və klassik MVC inteqrasiyası

---

##  Bu layihədən nə öyrənəcəksiniz?

✅ Arxitektura və qatlar arasında məsuliyyət bölgüsü  
✅ Domain, Application, Infrastructure və Presentation qatlarının qurulması  
✅ CQRS ilə əmrlər və sorğuların ayrılması  
✅ MediatR ilə handler-lərin idarə olunması  
✅ FluentValidation ilə giriş doğrulaması  
✅ AutoMapper ilə DTO və Entity-lərin map edilməsi  
✅ Exception middleware ilə qlobal istisna idarəçiliyi  
✅ Unit test yazmaq və kodun testability-sini artırmaq  
✅ CI/CD-ə ilk addım (GitHub Actions ilə)
